Настройка прокси

The best way to use a browser with ZAP is to launch it from ZAP. It will then be automatically configured to proxy through ZAP and to ignore certificate warnings.

If for any reason you are unable or unwilling to do that then you will need to configure your browser to use ZAP as a proxy.
By default, ZAP uses an Address of 'localhost' and a Port of '8080', but these can be changed via the Options > Network > Local Servers/Proxies screen. Инструкции для последних версий наиболее часто используемых браузеров:

Chrome (on Windows)

    нажмите Кнопка "Настройка и управление Google Chrome" (вверху справа)
    Выберите Пункт меню "Параметры .."
    нажмите Кнопка "Изменить настройки прокси"
    нажмите Кнопка "Настройки LAN"
    Следовать Раздел "Настройки локальной сети Windows" ниже
Примечание: Прокси localhost (и связанные адреса) с более новыми версиями Chrome (> = 72) аргумент командной строки --proxy-bypass-list=<-loopback> должно быть предоставлено.

Firefox (on Windows)

    Выберите Меню "Инструменты"
    Выберите Пункт меню "Параметры .."
    Выберите Панель "Общие"
    Прокрутите до Раздел "Сетевые настройки"
    нажмите Кнопка "Настройки ..."
    Выберите Радиокнопка "Ручная настройка прокси"
    Войдите в 'HTTP Proxy:' field the 'Address' you configured in the options screen
    Войдите в 'Port' field to the right of the 'HTTP Proxy' field the 'Port' you configured in the options screen. Убедитесь, что «SSL-прокси» также настроен, выбрав «Использовать этот прокси-сервер для всех протоколов» или задав соответствующие значения.
    нажмите Кнопка «ОК» настройки подключения
    нажмите Опции кнопки "ОК"
Примечание: Прокси localhost (и связанные адреса) с более новыми версиями Firefox (> = 67) предпочтение network.proxy.allow_hijacking_localhost (доступно через about:config страницу) должен быть установлен на истина.

Firefox (on Linux)

    Выберите Меню "Правка"
    Выберите Пункт меню «Настройки»
    Следовать инструкции выше из панели "Общие"

Firefox (в OS X)

    Выберите Меню "Firefox"
    Выберите Пункт меню "Настройки ..."
    Следовать инструкции выше из панели "Общие"

Internet Explorer 7 и 8

    Выберите Меню "Инструменты"
    Выберите Пункт меню "Свойства обозревателя"
    Выберите Вкладка "Подключения"
    нажмите Кнопка "Настройки LAN"
    Следовать Раздел "Настройки локальной сети Windows" ниже

Opera

    Выберите "Меню" в верхнем левом углу.
    Выберите Пункт меню «Настройки»
    Выберите Пункт меню «Настройки»
    Выберите 'Продвинутая вкладка
    Выберите Пункт «Сеть» в левом списке
    нажмите Кнопка "Прокси-серверы"
    Выберите Флажок HTTP
    Войдите в field to the right of 'HTTP' the 'Address' you configured in the options screen
    Войдите в HTTP 'Port' field the 'Port' you configured in the options screen
    нажмите Кнопка «ОК» прокси-серверов
    нажмите Кнопка настроек "ОК"

Safari (on OS X)

    нажмите Кнопка "Настройки Safari" в правом верхнем углу.
    Выберите Пункт меню «Настройки»
    Выберите 'Продвинутая вкладка
    нажмите Кнопка "Изменить настройки" рядом с надписью "Прокси".
    Следовать Раздел «Настройки системного прокси OS X» ниже

Safari (on Windows)

    нажмите Кнопка "Показать меню общих настроек Safari" в правом верхнем углу.
    Выберите Пункт меню «Настройки»
    Выберите 'Продвинутая вкладка
    нажмите Кнопка "Изменить настройки" рядом с надписью "Прокси".
    нажмите Кнопка "Настройки LAN"
    Следовать Раздел "Настройки локальной сети Windows" ниже

Настройки системного прокси OS X

    Выберите 'Web proxy (HTTP)'
    Войдите в 'Address:' field the 'Address' you configured in the options screen
    Войдите в 'Port' field the 'Port' you configured in the options screen
    нажмите Прокси-серверы Настройка кнопки «ОК»
    нажмите Кнопка «Применить» в параметрах сети

Linux Gnome Proxy settings

    нажмите 'Gnome Settings' in the top corner
    Выберите 'Network' item in the left
    Войдите в 'Proxy' in the 'Proxy' section
    нажмите 'Network Proxy' toggle
    Select 'Manual' in the 'Configuration' dropdown
    Войдите в HTTP (or HTTPs) Proxy 'URL' field the 'URL' you configured in the options screen
    Enter in the HTTP (or HTTPs) Proxy 'Port' field the 'Port' you configured in the options screen

Настройки локальной сети Windows

    Select the Флажок "Использовать прокси-сервер для вашей локальной сети"
    Enter in the 'Address:' field the 'Address' you configured in the options screen
    Enter in the 'Port' field the 'Port' you configured in the options screen
    Press the Кнопка «ОК» настройки локальной сети (LAN)
    Press the Кнопка "ОК" в настройках Интернета
