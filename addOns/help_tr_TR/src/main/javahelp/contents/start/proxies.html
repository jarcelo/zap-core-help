Proxy'leri yapılandırma

The best way to use a browser with ZAP is to launch it from ZAP. It will then be automatically configured to proxy through ZAP and to ignore certificate warnings.

If for any reason you are unable or unwilling to do that then you will need to configure your browser to use ZAP as a proxy.
By default, ZAP uses an Address of 'localhost' and a Port of '8080', but these can be changed via the Options > Network > Local Servers/Proxies screen. En sık kullanılan tarayıcıların en son sürümleri için talimatlar:

Chrome (Windows'ta)

    Tuşuna basın 'Google Chrome 'u özelleştirme ve denetleme' tuşu (sağ üst)
    Seçin 'Seçenekler..' menü öğesi
    Tuşuna basın 'vekil sunucu ayarlarını değiştir' tuşu
    Tuşuna basın 'LAN Ayarları' tuşu
    Takip et 'Windows LAN Ayarları' aşağıdaki bölüm
Note: To proxy localhost (and related addresses) with newer Chrome versions (>= 72) the command line argument --proxy-bypass-list=<-loopback> must be provided.

Firefox (Windows'ta)

    Seçin 'Araçlar' menüsü
    Seçin 'Seçenekler..' menüsü öğesi
    Select the 'General' panel
    Scroll to 'Network Settings' section
    Tuşuna basın 'Ayarlar...' düğmesi
    Seçin 'Manuel proxy yapılandırması' radyo düğmesi
    Girin 'HTTP Proxy:' field the 'Address' you configured in the options screen
    Girin 'Port' field to the right of the 'HTTP Proxy' field the 'Port' you configured in the options screen. Ensure 'SSL Proxy' is also configured, either by selecting 'Use this proxy server for all protocols' or by setting the corresponding values.
    Tuşa basın Bağlantı Ayarı 'OK' düğmesi
    Tuşa basın Seçenekler 'Tamam' düğmesi
Note: To proxy localhost (and related addresses) with newer Firefox versions (>= 67) the preference network.proxy.allow_hijacking_localhost (accessible through the about:config page) must be set to true.

Firefox (Linux'ta)

    Seçin'Düzenle' menüsü
    Seçin'Tercihler' menü öğesi
    Takip etinstructions above from the 'General' panel

Firefox (OS X'de)

    Seçin 'Firefox' menüsü
    Seçin 'Tercihler...' menü öğesi
    Follow the instructions above from the 'General' panel

Internet Explorer 7 ve 8

    Seçin 'Araçlar' menüsü
    Seçin 'İnternet Seçenekleri' menü öğesi
    Seçin 'Bağlantılar' sekmesi
    Tuşuna basın 'LAN Ayarları' düğmesi
    Takip et 'Windows LAN Ayarları' bölümü

Opera

    Seçin Sol üst köşedeki 'Menü'
    Seçin 'Ayarlar' menü öğesi
    Seçin 'Tercihler' menü öğesi
    Seçin 'Gelişmiş sekme
    Seçin Sol listedeki 'Ağ' öğesi
    Tuşuna basın 'Vekil Sunucuları' tuşu
    Seçin 'HTTP' kontrol kutusu
    Girin field to the right of 'HTTP' the 'Address' you configured in the options screen
    Girin HTTP 'Port' field the 'Port' you configured in the options screen
    Tuşuna basın Vekil sunucuları 'OK' tuşu
    Tuşuna basın Tercihler 'OK' tuşu

(OS X'te bulunan) Safari

    Tuşuna basın Sağ üst köşedeki 'Safari ayarları' düğmesi
    Seçin 'Tercihler' menü öğesi
    Seçin 'Gelişmiş' sekme
    Tuşuna basın 'Vekil sunucular' etiketinin hemen yanındaki 'Ayarları Değiştir' tuşuna tıklayın
    Takip et Aşağıdaki 'OS X Sistem Proxy ayarları' bölümü

Safari (Windows'ta)

    Tuşuna basın Sağ üst köşedeki 'Genel Safari ayarlarının bir menüsünü görüntüle' düğmesi
    Seçin 'Tercihler' menü öğesi
    Seçin 'Gelişmiş' sekme
    Tuşuna basın 'Vekil sunucular' etiketinin yanındaki 'Ayarları Değiştir' tuşuna tıklayın
    Tuşuna basın 'Lan Ayarları' tuşu
    Takip et 'Windows LAN Ayarları' bölümü

OS X Sistem Proxy ayarları

    Seçin 'Web proxy (HTTP)'
    Girin 'Address:' field the 'Address' you configured in the options screen
    Girin 'Port' field the 'Port' you configured in the options screen
    Tuşuna basın Vekil sunucular Ayarı 'OK' tuşu
    Tuşuna basın Ağ Seçenekleri 'Uygula' tuşu

Linux Gnome Proxy settings

    Tuşuna basın 'Gnome Settings' in the top corner
    Seçin 'Network' item in the left
    Girin 'Proxy' in the 'Proxy' section
    Tuşuna basın 'Network Proxy' toggle
    Select 'Manual' in the 'Configuration' dropdown
    Girin HTTP (or HTTPs) Proxy 'URL' field the 'URL' you configured in the options screen
    Enter in the HTTP (or HTTPs) Proxy 'Port' field the 'Port' you configured in the options screen

Windows LAN Ayarları

    Select the 'LAN 'ınız için bir proxy sunucusunu kullanın' onay kutusu
    Enter in the 'Address:' field the 'Address' you configured in the options screen
    Enter in the 'Port' field the 'Port' you configured in the options screen
    Press the Yerel Bölge Ağı (LAN) Ayarı 'Tamam' butonu
    Press the İnternet Seçenekleri 'OK' tuşu
