HTTP Oturumları

This tool keeps track of the existing HTTP Sessions on a particular Site and allows the ZAP user to force all requests to be on a particular session. Temel olarak, kullanıcının kolayca bir Sitedeki kullanıcı oturumları arasında geçiş yapmak ve yeni bir oturum oluşturmak mevcut olanları "yok etmeden".

The tool is only intended for manual testing and should not be used in automation - there are much better alternatives. If you are new to ZAP automation then the best place to start is the ZAP Authentication Decision Tree (external link).

Bu, HTTP İletisi olan Oturum Tokenleri kavramına dayanır bir HTTP sunucusuna izin veren parametreler (yalnızca Çerezler için) depolanan önceki istekler veya verilerle bir istek mesajı bağlayın. In the case of ZAP, conceptually, session tokens have been classified into 2 categories: default session tokens and site session tokens. Varsayılan oturum belirteci, kullanıcının ayarlayabileceği simgelerdir. Seçenekleri'nde ekran 'ı içerir ve varsayılan olarak otomatik olarak değerlendirilen belirteçlerdir herhangi bir site için oturum belirteci (ör. phpsessid, jsessionid, vb.). Site Oturum belirteçleri, belirli bir site için belirteçler kümesidir ve genellikle Params Tab 'da bulunan pop-up menüler kullanılarak kurulur.

Bu araç, tanımlı oturum özelliklerini kullanarak otomatik olarak algılar otomatik olarak algılanan varsayılan oturum özellikleri, herhangi bir HTTP oturumu ile iletişim içinde olanlardır. Tespit edilen oturumlar, HTTP Oturumları Sekmesi .

Kullanıcı HTTP Oturumu Sekmesi 'nde bulunan düğmeyi kullanarak, mevcut birisini yok etmeden yeni bir oturum açmak veya oturumlar 'aktif' olarak nitelendirildi. Bir oturum "etkin" olduğunda, tüm giden ilgili Siteye gönderilen talepler değiştirilir, oturum Etkin oturumu eşleştirmek için ayarlanmış belirteçleri.   Bağlam | Bağlam İste. Bu şekilde, kullanıcı bazı iletileri belirli bir oturumun bir parçası haline getirmeye zorlayabilir ve sonra başka bir oturuma mesajlar atın ve gönderin.

HTTP Oturumları aracı, HTTP Oturumu kullanılarak yapılandırılmıştır Seçenekler ekranı .
