Sessões Http

This tool keeps track of the existing HTTP Sessions on a particular Site and allows the ZAP user to force all requests to be on a particular session. Basicamente ela permite que se possa alternar facilmente entre as sessões de usuário em um Site e criar uma nova sessão sem "destruir" as já existentes.

The tool is only intended for manual testing and should not be used in automation - there are much better alternatives. If you are new to ZAP automation then the best place to start is the ZAP Authentication Decision Tree (external link).

É baseada no conceito de Tokens de sessão - parâmetros de mensagem HTTP (agora apenas Cookies), que permitem a um servidor HTTP conectar uma mensagem de request com quaisquer requests ou dados armazenados anteriormente. In the case of ZAP, conceptually, session tokens have been classified into 2 categories: default session tokens and site session tokens. Os tokens de sessão padrão são aquelas que o usuário pode definir na tela de opções e que são automaticamente considerados tokens de sessão para qualquer site (por exemplo PHPSESSID, jsessionid, etc). Os tokens de sessão de sites são um conjunto de tokens para um determinado site e geralmente são criados usando-se os menus pop-up disponíveis na Aba Params.

Esta ferramenta detecta automaticamente qualquer sessão HTTP que existe na comunicação, usando os tokens de sessão definida ou os tokens de sessão padrão automaticamente detectados. As sessões detectadas são mostradas na Guia de sessões Http.

O usuário pode, usando o botão disponível na guia sessões HTTP, criar uma nova sessão sem destruir a existente, ou pode forçar uma das sessões como 'ativa'. Quando uma sessão está 'ativa', todas as solicitações de saída enviadas para o Site correspondente são modificadas, os tokens de sessão sendo configurados para corresponder à sessão ativa. Desta forma, o usuário pode facilmente forçar algumas mensagens para ser 'parte' de uma sessão e depois alternar e enviar mensagens em outra sessão.

A ferramenta de sessões Http é configurada usando a tela de opções de sessões Http.
