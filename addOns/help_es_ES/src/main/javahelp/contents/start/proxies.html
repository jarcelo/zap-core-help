Configurando proxies

The best way to use a browser with ZAP is to launch it from ZAP. It will then be automatically configured to proxy through ZAP and to ignore certificate warnings.

If for any reason you are unable or unwilling to do that then you will need to configure your browser to use ZAP as a proxy.
By default, ZAP uses an Address of 'localhost' and a Port of '8080', but these can be changed via the Options > Network > Local Servers/Proxies screen. Las instrucciones para las últimas versiones de los navegadores más utilizados son:

Chrome (en Windows)

    Pulse el botón ( arriba a la derecha ) 'Personalizar y controlar Google Chrome'
    Seleccione el elemento de menú 'Opciones...'
    Pulse el botón 'Cambiar la configuración del proxy'
    Pulse el botón 'Configuración de LAN'
    Siga la sección 'Configuración LAN de Windows'
Nota: Para proxificar localhost (y direcciones relacionadas) con versiones más nuevas de Chrome (>= 72) el parámetro en la línea de comandos --proxy-bypass-list=<-loopback> deberá ser provisto.

Firefox (en Windows)

    Seleccione el menú 'Herramientas'
    Seleccione el elemento de menú 'Opciones...'
    Seleccione Panel 'General'
    Desplácese hasta Sección 'Configuración de red'
    Pulse el botón 'Configuración...'
    Seleccione la opción 'Configuración manual del proxy'
    Ingrese El 'Proxy HTTP:' el campo 'Dirección' que configuró en la pantalla de opciones
    Ingrese El campo 'Puerto' a la derecha del campo 'Proxy HTTP' el 'Puerto' que configuró en la pantalla de opciones. Asegúrese de que 'Proxy SSL' también esté configurado, ya sea seleccionando 'Usar este servidor proxy para todos los protocolos' o configurando los valores correspondientes.
    Pulse el botón 'OK'
    Pulse el botón 'OK'
Nota: Para proxificar localhost (y direcciones relacionadas) con versiones más nuevas de Firefox (>= 67) la preferencia network.proxy.allow_hijacking_localhost (accesible desde la página about:config debe establecerse a true.

Firefox (en Linux)

    SeleccioneMenú de 'Editar'
    SeleccioneObjeto del menu de 'Preferencias'
    SigaLas instrucciones anteriores desde el panel 'General'

Firefox (en OS X)

    Seleccione Menú 'Firefox'
    Seleccione El elemento de menú "Preferencias..."
    Siga Las instrucciones anteriores desde el panel 'General'

Internet Explorer 7 y 8

    Seleccione Menú de 'Herramientas'
    Seleccione el elemento de menú "Opciones de Internet"
    Seleccione la pestaña "Conexiones"
    Pulse el botón "Configuración de LAN"
    Siga La sección "Configuración LAN de Windows" más abajo

Opera

    Seleccione "Menú" en la esquina superior izquierda
    Seleccione Elemento de menú "Configuración"
    Seleccione Elemento de menú "preferencias"
    Seleccione Pestaña "avanzado"
    Seleccione Elemento "red" en la lista de la izquierda
    Pulse Botón de "Servidores proxy"
    Seleccione La casilla 'HTTP'
    Ingrese El campo a la derecha de 'HTTP' la 'Dirección' que configuró en la pantalla de opciones
    Ingrese El 'Puerto' HTTP en el campo 'Puerto' que configuró en la pantalla de opciones
    Pulse El botón 'OK' en Servidores Proxy
    Pulse El botón 'OK' en Preferencias

Safari (en OS X)

    Pulse El botón 'Configuración de Safari' en la esquina superior derecha
    Seleccione el elemento de menú 'Preferencias'
    Seleccione la pestaña 'Avanzado'
    Pulse el botón 'Cambiar configuraciones' a continuación de la etiqueta 'Proxies'
    Siga La Sección 'Configuración de proxy del sistema OS X' a continuación

Safari (en Windows)

    Pulse El botón 'Mostrar un menú de configuración general de Safari' en la esquina superior derecha
    Seleccione Elemento de menú 'preferencias'
    Seleccione la pestaña 'avanzado'
    Pulse El botón de 'cambiar configuraciónes' al lado de la etiqueta 'proxis'
    Pulse Botón "Configuración LAN"
    Sigua la La sección "Configuración LAN de Windows" más abajo

Configuración de Proxy del sistema OS X

    Seleccione 'Web proxy (HTTP)'
    Introduzca El campo 'Dirección:' la 'Dirección' que configuró en la pantalla de opciones
    Introduzca El campo 'Puerto' el 'Puerto' que configuró en la pantalla de opciones
    Pulse Botón 'OK' de la configuración de servidores proxy
    Pulse Botón 'Aplicar' en opciones de red

Linux Gnome Proxy settings

    Presione el 'Gnome Settings' in the top corner
    Seleccione 'Network' item in the left
    Introduzca en 'Proxy' in the 'Proxy' section
    Presione el 'Network Proxy' toggle
    Select 'Manual' in the 'Configuration' dropdown
    Introduzca en HTTP (or HTTPs) Proxy 'URL' field the 'URL' you configured in the options screen
    Enter in the HTTP (or HTTPs) Proxy 'Port' field the 'Port' you configured in the options screen

Configuración LAN de Windows

    Select the La casilla de 'Usar un servidor Proxy para su servidor LAN'
    Enter in the El campo 'Dirección:' la 'Dirección' que configuró en la pantalla de opciones
    Enter in the El campo 'Puerto' el 'Puerto' que configuró en la pantalla de opciones
    Press the Botón 'OK' de la configuración de Red de Área Local (LAN)
    Press the Botón 'OK' de las Opciones de Internet
