扫描策略对话框

This allows you to enable and disable the rules that are run when performing an active scan.
The first screen allows you to define the default levels as well as the levels for all of the rules in a specific category.

The category screens allow you to define the levels for every individual rule.

阈值

这控制了ZAP报告潜在漏洞的可能性。
  • If you select Off then the scan rule won't run.
  • 如果您选择“低”’这个选项，则会引发更多潜在的问题。这可能会导致误报数量的增加。
  • 如果选择“高”，潜在的问题则会被提高而一些真正的问题可能会被遗漏 （或漏报）。

强度

这会控制ZAP即将开始进行攻击的次数。
如果您选择“低”这个选项，那将会使用更少的攻击。虽然会更快但可能会遗漏一些问题。
如果您选择“高”，则会使用更多的攻击。虽然可能会发现更多的问题但会花费更多的时间。
“疯狂”的级别通常只应用于应用程序的小部分，它可能会导致大量的攻击而且会占用相当长的时间。

Note: Please be aware that use of the moniker "Insane" with regard to scan strength is simply a name chosen to represent the most extreme strength of scanning, it is not a commentary or reference to mental health or personal stability.
