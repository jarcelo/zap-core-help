ZAP by Checkmarx Desktop User Guide

Welcome to the Zed Attack Proxy (ZAP) Desktop User Guide.

This is available both as context sensitive help within ZAP and online at https://www.zaproxy.org/docs/desktop/

ZAP adalah alat uji penetrasi terintegrasi yang mudah digunakan untuk menemukan kerentanan dalam aplikasi web.

Hal ini dirancang untuk digunakan oleh orang-orang dengan berbagai macam keamanan dan pengalaman seperti itu adalah tempat ideal untuk pengembang dan fungsional penguji yang baru untuk pengujian penetrasi.

ZAP menyediakan pemindai otomatis sekaligus seperangkat alat yang memungkinkan kamu menemukan kerentanan keamanan secara manual.

ZAP can also be run in a completely automated way - see the ZAP website for more details.

If you are new to ZAP then its recommended that you look at the Getting Started section.

ZAP is a fork of the open source variant of the Paros Proxy.
