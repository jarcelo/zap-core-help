Mengkonfigurasi Proxy

The best way to use a browser with ZAP is to launch it from ZAP. It will then be automatically configured to proxy through ZAP and to ignore certificate warnings.

If for any reason you are unable or unwilling to do that then you will need to configure your browser to use ZAP as a proxy.
By default, ZAP uses an Address of 'localhost' and a Port of '8080', but these can be changed via the Options > Network > Local Servers/Proxies screen. Instruksi untuk versi terbaru dari peramban yang paling umum digunakan:

Chrome (pada Windows)

    Tekan Tombol 'Menyesuaikan dan kontrol Google Chrome' (kanan atas)
    Pilih Item menu 'Pilihan..'
    Tekan Tombol 'Ubah pengaturan proxy'
    Tekan Tombol 'Pengaturan LAN'
    Ikuti Bagian di bawah ini 'Pengaturan LAN Windows'
Note: To proxy localhost (and related addresses) with newer Chrome versions (>= 72) the command line argument --proxy-bypass-list=<-loopback> must be provided.

Firefox (pada Windows)

    Pilih lah Menu 'Tools'
    Pilih Item menu 'Pilihan..'
    Select the 'General' panel
    Scroll to 'Network Settings' section
    Tekan Tombol 'Setelan...'
    Pilih petunjuk manual untuk konfigurasi tombol radio
    Masukan di 'HTTP Proxy:' field the 'Address' you configured in the options screen
    Masukkan dalam 'Port' field to the right of the 'HTTP Proxy' field the 'Port' you configured in the options screen. Ensure 'SSL Proxy' is also configured, either by selecting 'Use this proxy server for all protocols' or by setting the corresponding values.
    Tekan Pengaturan koneksi tombol 'OK'
    Tekan Pilihan tombol 'OK'
Note: To proxy localhost (and related addresses) with newer Firefox versions (>= 67) the preference network.proxy.allow_hijacking_localhost (accessible through the about:config page) must be set to true.

Firefox (pada Linux)

    PilihMenu 'Sunting'
    PilihPilihItem menu 'Preferensi'
    Ikutiinstructions above from the 'General' panel

Firefox (pada OS X)

    Pilih Menu 'Firefox'
    Pilih 'Preferensi...' item menu
    Follow the instructions above from the 'General' panel

Internet Explorer 7 dan 8

    Pilih Menu 'Tools'
    Pilih Menu item 'Pilihan Internet'
    Pilih Tab 'Koneksi'
    Tekan Tombol 'Pengaturan LAN'
    Ikuti 'Windows LAN Settings' bagian bawah

Opera

    Pilih 'Menu' di sudut atas sebelah kiri
    Pilih Item menu 'Pengaturan'
    Pilih 'Preferensi' item menu
    Pilih Memajukan Tombol
    Pilih 'Jaringan' item di tangan kiri daftar
    Tekan tombol 'Proxy Servers'
    Pilih kotak centang 'HTTP'
    Masukkan di field to the right of 'HTTP' the 'Address' you configured in the options screen
    Masukkan di HTTP 'Port' field the 'Port' you configured in the options screen
    Tekan Proxy server tombol 'OK'
    Tekan Preferensi tombol 'OK'

Safari (pada OS X)

    Tekan 'Setelan Safari' tombol di sudut kanan atas
    Pilih 'Preferensi' item menu
    Pilih 'Advanced' tab
    Tekan Tombol 'Ubah Setelan' di sebelah label 'Proxy'
    Ikuti 'OS X Sistem pengaturan Proxy' bagian bawah

Safari (pada Windows)

    Tekan 'Menampilkan menu general setelan Safari' tombol di sudut kanan atas
    Pilih 'Preferensi' item menu
    Pilih Tab 'Lanjutan'
    Tekan 'Mengubah Pengaturan' tombol di sebelah 'Proxy' label
    Tekan Tombol 'Pengaturan LAN'
    Ikuti 'Windows LAN Settings' bagian bawah

OS X Sistem pengaturan Proxy

    Pilih 'Web proxy (HTTP)'
    Masukkan di 'Address:' field the 'Address' you configured in the options screen
    Masukkan di 'Port' field the 'Port' you configured in the options screen
    Tekan Proxy Setting tombol 'OK'
    Tekan Jaringan Pilihan 'Terapkan' tombol

Linux Gnome Proxy settings

    Tekan 'Gnome Settings' in the top corner
    Pilih 'Network' item in the left
    Masukkan di 'Proxy' in the 'Proxy' section
    Tekan 'Network Proxy' toggle
    Select 'Manual' in the 'Configuration' dropdown
    Masukkan di HTTP (or HTTPs) Proxy 'URL' field the 'URL' you configured in the options screen
    Enter in the HTTP (or HTTPs) Proxy 'Port' field the 'Port' you configured in the options screen

Setelan LAN Windows

    Select the kotak centang 'Gunakan server proxy untuk LAN anda'
    Enter in the 'Address:' field the 'Address' you configured in the options screen
    Enter in the 'Port' field the 'Port' you configured in the options screen
    Press the Tombol 'OK' Pengaturan Local Area Network (LAN)
    Press the Tombol 'OK' Pilihan Internet
