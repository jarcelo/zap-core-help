ZAP by Checkmarx Desktop User Guide

Welcome to the Zed Attack Proxy (ZAP) Desktop User Guide.

This is available both as context sensitive help within ZAP and online at https://www.zaproxy.org/docs/desktop/

Ang ZAP ay isang simple para sa paggamit ng sa isinamang paglagpas ng paggamit na kasangkapan para sa paghahanap ng kahinaan sa web ng mga aplikasyon.

Ito ay nakadesenyo upang magamit ng mga tao kasama ang isang malawak na sakop ng karanasan ng seguridad at gaya ng uliran para sa mga developer aat tagapagsubok ng pagpapatakbo na baguhan para sa paglagpas ng pagsubok.

Ang mga awtomatikong naibigay ng ZAP sa mga iskner pati na ang isng ayos ng mga kasanagkapan na pinahihintulutan ka na hanapin ang seguridad ng mga kahinaan ng manu-mano.

ZAP can also be run in a completely automated way - see the ZAP website for more details.

If you are new to ZAP then its recommended that you look at the Getting Started section.

ZAP is a fork of the open source variant of the Paros Proxy.
